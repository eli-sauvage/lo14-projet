# lo14-projet
## To start the server
`cd lo14-projet/server && bash server.sh PORT` on the server machine
## To use as client
### Install
`cd lo14-projet && bash setup.sh` and follow instructions
### Usage
`vsh CMD ADRESS PORT [opt]` on the client
